An organization dedicated to reviving the forgotten device of the Motorola-STSM family -- the DROID 3 and MILESTONE 3.

# Currently supported
- Safestrap 3.75 (TWRP 2.7.1.0) for DROID 3 and MILESTONE 3
- LineageOS 11 (Android 4.4.4 KitKat) on DROID 3 and MILESTONE 3
- CyanogenMod 12.1 (Android 5.1.1 Lollipop) on DROID 3 and MILESTONE 3
- crDroid 3 (Android 7.1.2 Nougat) on DROID 3 and MILESTONE 3
